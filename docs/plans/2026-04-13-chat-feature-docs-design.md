# Design Doc: Chat Feature Documentation

## 1. Goal
vCenter Event Assistant の主要機能の1つである「チャット機能」について、独立したドキュメント `docs/chat.md` を追加する。
ユーザーが期間を指定し、イベントやメトリクスをコンテキストとして与えて質問できること、およびプロンプトスニペットの設定・活用方法を網羅する。

## 2. Structure of `docs/chat.md`

### 1. チャット機能の概要
- 目的と主なメリット（vCenter データの分析を AI で支援）

### 2. 具体的な活用例（ユースケース）
- **例1：異常検知後の原因調査**: 最近のイベントとホスト負荷の相関分析
- **例2：週次レポートの深掘り**: 特定時期のイベント傾向の要約

### 3. 基本的な使い方
- **ステップ 1: 期間の選択**: 分析対象時間の指定
- **ステップ 2: コンテキスト（情報）の選択**: イベント・メトリクスの ON/OFF
- **ステップ 3: 質問の入力と送信**: 対話インターフェースの使い方

### 4. プロンプトスニペットの活用
- スニペットとは：定型的な質問の保存
- 設定方法：設定タブ ＞ チャットサブタブ
- 呼び出し方：チャットパネルのボタンから実行

### 5. 注意事項
- トークン制限と匿名化設定について

## 3. Implementation Approach
- 新規ファイル `docs/chat.md` を作成する。
- 既存の `README.md` や `docs/frontend.md` からこの新ドキュメントへのリンクを追加する。
- `docs/images/chat.png` や `docs/images/settings-chat.png` を適切に埋め込む。

## 4. Verification Plan
- 作成した Markdown ファイルが正しくレンダリングされるか確認する。
- リンク切れがないか確認する。
