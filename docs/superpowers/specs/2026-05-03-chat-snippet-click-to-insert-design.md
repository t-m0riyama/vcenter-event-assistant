# チャットスニペット: クリック即時下書き追記

## 目的

チャットパネルのサンプルプロンプト（スニペット）を、クリックした時点で下書き（textarea）の末尾へ追記する。送信は行わない。従来の「複数選択 → 下書きに挿入」の二段操作を廃止する。

## 仕様

- スニペットボタンのクリック = 当該行の `text` を下書き末尾へ即時追記する。
- 追記時の区切りは既存と同様: 下書きが空（または空白のみ）なら本文のみ。非空なら下書き末尾を `trimEnd` したうえで `\n\n` を挟んで本文を連結する。
- 本文は `trim` した結果が空なら追記しない（下書きは不変）。
- 同一スニペットを再度クリックした場合も、その都度末尾へ追記する（重複可）。
- 「下書きに挿入」ボタン、選択状態（`aria-pressed` 等）は撤去する。
- 送信中（`loading`）はスニペットボタンを無効化する（現行と同様）。
- クリック後に textarea へフォーカスを移さない（連続クリックを想定）。

## 実装の境界

- 変更対象はフロントのチャットパネルと純関数ヘルパー、関連テストに限定する。
- スニペットの CRUD・ストレージ・設定画面は変更しない。

## テスト方針

- 純関数 `appendChatSampleTextToDraft` を単体テストで検証する。
- `ChatPanel` の結合テストで、クリック順が下書きに反映されること、送信中にスニペットが無効になることを検証する。
