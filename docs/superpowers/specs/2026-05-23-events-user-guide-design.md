# イベントタブ利用者向けドキュメント — 設計

**日付:** 2026-05-23  
**ステータス:** 承認済み（実装に反映）  
**正本（利用者向け）:** [`docs/user-guides/events.md`](../../user-guides/events.md)

## 目的

メインタブ **イベント** の一覧・絞り込み・運用メモ・CSV 出力を利用者向け正本にまとめる。

## スコープ

- ツールバー（最小スコア、表示件数、ページング、CSV）
- 絞り込み条件（期間、種別・重大度・メッセージ・コメント）
- 表の列、ガイド、運用メモ PATCH
- 自動更新、概要・チャットとの違い

## 受け入れ条件

- [x] UI ラベルと一致
- [x] 既定は期間未指定＝DB 全件対象と明記
- [x] 運用メモはサーバー保存と明記
