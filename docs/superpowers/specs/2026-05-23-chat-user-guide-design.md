# チャット機能利用者向けドキュメント — 設計

**日付:** 2026-05-23  
**ステータス:** 承認済み（実装に反映）  
**正本（利用者向け）:** [`docs/user-guides/chat.md`](../../user-guides/chat.md)

## 目的

期間コンテキスト付き **LLM チャット** の画面操作・データの送られ方・制限事項を、[`digests.md`](../../user-guides/digests.md) 等と同型で利用者向け正本にまとめる。既存 [`docs/chat.md`](../chat.md) は概要・活用例の補助とし、先頭から正本へ誘導する。

## スコープ

### 含める

- チャットの目的と全体像（イベント集約は常に含む、メトリクスは任意）
- 集計期間・vCenter・期間メトリクス・閾値
- 送信・プレビュー・会話クリア・コピー・Enter 操作
- ブラウザ localStorage（会話・下書き・サンプル・最大保持件数）
- LLM 未設定時・トークン目安・匿名化（概要）
- ダイジェスト・アラート・タイムラインとの違い
- 設定 → チャット（サンプル質問 JSON）
- FAQ

### 含めない

- LangChain / LangSmith の実装詳細
- `LLM_CHAT_*` の全環境変数一覧（getting-started / backend）
- Copilot CLI のセットアップ手順の詳細

## 受け入れ条件

- [x] 現行 UI（メトリクス 4 チェック、プレビュー、20 ターン LLM 送信）と一致
- [x] 「イベント用チェックボックス」など旧 UI の記述をしない
- [x] 会話はサーバー非保存と明記
